# Overview

The Knowledge Base is where you teach the AI Receptionist about your business. By uploading documents, the AI learns your policies, pricing, and services so it can answer customer questions accurately.

# Step-by-Step Instructions

### Step 1: Go to Knowledge Base

Click **Knowledge** in the left menu.

### Step 2: Upload a File

Find the area that says **Upload New Document**. 

You can click the box to select a file from your computer, or you can drag and drop a file directly into the box.

We support the following file types:
- PDF documents
- Word documents (.docx)
- Text files (.txt)
- Spreadsheets (.csv)

### Step 3: Wait for Processing

Once you select a file, the system will process it. You will see it appear in the list of your active documents. 

The AI can now read this document and use the information inside it to answer questions!

# Tips for Good Documents

- **Keep it Simple:** The clearer your document is written, the better the AI will understand it.
- **Use Q&A Format:** If you have a list of Frequently Asked Questions, save it as a PDF or Text file and upload it. The AI loves reading FAQs.
- **Keep it Updated:** If your pricing changes, remember to delete the old pricing document and upload the new one. This prevents the AI from getting confused.
