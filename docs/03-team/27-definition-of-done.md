# Definition of Done: Team Management

The module is considered complete when:
- [x] Staff members can be created, updated, and soft-deleted.
- [x] The "Out of Bounds" validation correctly prevents staff from working outside Organization hours.
- [x] Dashboard Invites successfully deliver an email, and the magic link correctly links the `userId` to the `staffId` upon signup.
- [x] Staff can configure one-off exceptions (sick days) seamlessly.
- [x] The UI explicitly differentiates between a Staff Profile (calendar object) and a Dashboard User (authentication object).
- [x] All 9 test cases in `25-test-cases.md` pass automatically or manually in Staging.
