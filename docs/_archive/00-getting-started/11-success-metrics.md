# Success Metrics

## Business KPIs
- **Monthly Recurring Revenue (MRR)**: Total predictable revenue generated by active subscriptions.
- **Customer Acquisition Cost (CAC)**: The cost to acquire a new paying Organization.
- **Lifetime Value (LTV)**: Projected total revenue from a single Organization over their lifespan on the platform.
- **Net Revenue Retention (NRR)**: Revenue retained from existing customers over a given period, including expansion and downgrades/churn. Target > 110%.
- **Churn Rate**: The percentage of Organizations that cancel their subscription monthly. Target < 2%.

## AI KPIs
- **Deflection Rate**: Percentage of conversations successfully resolved by the AI without human intervention. Target > 85%.
- **Booking Conversion Rate**: Percentage of conversations displaying booking intent that result in a confirmed appointment. Target > 60%.
- **Hallucination Rate**: Percentage of AI responses containing factually incorrect information based on the provided Knowledge Base. Target effectively 0%.
- **Intent Recognition Accuracy**: Percentage of user messages accurately classified by the NLU engine. Target > 95%.

## Platform KPIs
- **Appointments Booked**: Total number of appointments facilitated by the platform globally.
- **Active Organizations (WAU/MAU)**: Organizations that log into the dashboard or have the AI actively handling conversations weekly/monthly.
- **Time to Value (TTV)**: The duration between an Organization signing up and their AI Agent successfully booking its first appointment. Target < 1 hour.

## Performance KPIs
- **Time to First Byte (TTFB)**: Server response time for critical API endpoints. Target < 200ms.
- **AI Latency**: The time from receiving a user message webhook to sending the AI's response webhook. Target < 1000ms.
- **Sync Latency**: Time to reflect external calendar events in the platform's availability engine. Target < 5s.
- **Frontend Core Web Vitals**: Largest Contentful Paint (LCP) < 2.5s, First Input Delay (FID) < 100ms, Cumulative Layout Shift (CLS) < 0.1.

## Operational KPIs
- **Support Ticket Volume**: Number of technical support requests generated per 100 Active Organizations.
- **Time to Resolution (TTR)**: Average time to resolve a customer support ticket.
- **Deployment Frequency**: Number of successful production deployments per week.
- **Change Failure Rate**: Percentage of deployments causing a degradation in service. Target < 1%.
