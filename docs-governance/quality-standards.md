# Quality Standards

Documentation that fails these standards will be strictly rejected.

## Automatic Rejection Criteria
- **Incomplete**: Sections marked "TBD" or containing placeholder text (Lorem Ipsum).
- **Inconsistent Terminology**: Using terms not defined in `taxonomy.json`.
- **Broken Links**: Internal or external 404s.
- **Missing Metadata**: Lacking required frontmatter fields (Title, Description, Category).
- **Missing Owner**: Frontmatter lacks a defined owner.
- **Missing Review Date**: Frontmatter lacks a `last_reviewed` timestamp.
- **Missing Related Documents**: No cross-references provided for complex topics.
