# Review Checklists

Consolidated checklist for the Documentation Reviewer Agent.

## 1. Structural Checklist
- [ ] Follows `folder-conventions.md`.
- [ ] Filenames follow `naming-conventions.md`.
- [ ] Metadata is complete and valid.

## 2. Content Checklist
- [ ] Adheres to `style-guide.md` (Active voice, sentence case headings).
- [ ] No placeholders or TODOs.
- [ ] Terminology matches the platform UI exactly.

## 3. Relational Checklist
- [ ] Links to prerequisite knowledge exist.
- [ ] Links to next steps or related modules exist.
- [ ] No circular dependencies in cross-references.
