# Folder Conventions

The physical directory structure maps directly to the site navigation.

## Hierarchy
- Top-level folders represent primary categories (e.g., `01-getting-started`, `02-architecture`).
- Sub-folders represent logical groupings within a category.
- A directory must contain an `index.mdx` file if it serves as a category overview.

## Media
- Images and diagrams must be stored in a localized `_assets/` directory within the relevant top-level folder, or a global `public/docs-assets/` for shared resources.
