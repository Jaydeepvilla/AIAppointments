# Naming Conventions

## File and Directory Names
- Must use `kebab-case`.
- All lowercase.
- No spaces or special characters.
- Prefix with sequential numbers if order matters (e.g., `01-getting-started`).
- Suffix with `.md` or `.mdx`.

## URL Slugs
- Derived directly from the file name minus the sequential prefix (e.g., `01-getting-started.md` becomes `/getting-started`).

## Headings
- Must use Sentence case.
- Must be concise and descriptive.
